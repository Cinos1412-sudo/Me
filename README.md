# Cinos

Site "hero" plein écran pour Cinos — développeur malgache passionné de tech,
jeux vidéo, manga et échecs. Vidéo de fond en boucle, navigation
glassmorphique et typographie cinématique, avec une page Portfolio dédiée.

## Pages

- `/` — Hero Cinos (vidéo de fond, nav glass, typographie cinématique)
- `/portfolio` — Portfolio (Navbar, Hero, About, Recent Works, Services, Footer)

Le lien "Portfolio" dans la nav de la page d'accueil mène vers `/portfolio`.
Un petit lien "← Cinos" en bord d'écran sur la page portfolio permet de revenir.

## Stack

- React + Vite + TypeScript
- Tailwind CSS
- react-router-dom (navigation entre les deux pages)
- framer-motion + lucide-react (composants du portfolio)

## Démarrer

```bash
npm install
npm run dev
```

Puis ouvrir http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
